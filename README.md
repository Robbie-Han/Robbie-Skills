# Robbie-Skills

这是一个 AI Coding Skills 技能库，包含了可以在 Claude Code 等 AI 编程工具中使用的各种技能模块。

## 简介

本仓库收集和管理了一系列可复用的 AI 编程技能（Skills），这些技能可以帮助 AI 助手更高效地完成特定的编程任务。每个技能都是一个独立的模块，包含了完整的文档、脚本和配置文件。

## 适用场景

这些 Skills 可以应用于：

- **Claude Code**: Anthropic 的 AI 编程助手
- **GitHub Copilot**: GitHub 的 AI 代码补全工具
- **Cursor**: AI 驱动的代码编辑器
- **其他支持自定义 AI 技能的开发工具**

## Skills 列表

### article-saver
文章保存工具，可以帮助保存和管理网络文章。

## 使用方法

1. 克隆本仓库到本地
2. 选择需要的 Skill 目录
3. 根据各 Skill 的 README.md 或 SKILL.md 文档进行配置和使用
4. 在支持的 AI Coding 工具中引用或加载相应的技能

- 清晰的文档说明
- 必要的脚本和配置文件
- 使用示例
